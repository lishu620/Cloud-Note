使用NPM安装Vue插件

<!-- truncate -->

## 正文

在准备使用Element开发网页的时候，官方文档提到[使用Vue开发](https://github.com/ElementUI/vue-cli-plugin-element)

因此选择在全局安装Vue插件

```
npm install -g @vue/cli
```

### 创建项目并添加 Element 插件

```
vue create my-app
cd my-app
vue add element
```
